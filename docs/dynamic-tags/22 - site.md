The `Site` tag gets site settings.

Example:

```html
<Site url />
```

## Fields

- `name` - Site name
- `description` - Site description
- `url` - Site URL

If any other field name is passed, it will return the same as  [The `Setting` tag](/docs/dynamic-tags/setting).
